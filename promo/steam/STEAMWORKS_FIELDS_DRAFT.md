## 5. 推薦 Steam 標籤

Steam 後台最多可填 **20 個標籤**，建議優先填入核心標籤，再補備用標籤。

### 核心標籤（B-02 製作人決定：全採用 + 額外新增 2 個）

| 標籤 | 理由 |
| --- | --- |
| Roguelite | 主類型，每局重開、局外成長 |
| Dice | 核心玩法機制 |
| Deckbuilding | 遺物構築與流派選擇 |
| Strategy | 重骰判斷與牌型選擇帶來策略深度 |
| Turn-Based | 回合制戰鬥 |
| Score Attack | 傷害倍率堆疊核心 |
| Replay Value | 高重玩性定位 |
| Singleplayer | 單人遊戲 |
| Casual | 短局、低上手門檻 |
| Indie | 獨立開發 |
| **小遊戲（Mini Game / Short）** | 製作人新增：輕度放鬆遊玩的小遊戲定位 |
| **僅滑鼠（Mouse only）** | 製作人新增：只需滑鼠操作，無鍵盤需求 |

### 備用標籤（B-03 製作人決定：全補入）

| 標籤 | 適合場景 |
| --- | --- |
| Card Game | 牌型判斷機制類似撲克牌 |
| Tabletop | 骰子桌游風格 |
| 2D | 2D 美術風格 |
| Cute | 美術風格偏可愛 |

## 7. 系統需求草稿

本遊戲以 Electron（Chromium 核心）包裝，硬體需求極低。

### Windows

| 項目 | 最低需求 | 建議需求 |
| --- | --- | --- |
| 作業系統 | Windows 10 64-bit | Windows 10/11 64-bit |
